# Conclave Frontend

The frontend interface for the Conclave peer review protocol.

## Overview
A React + Vite + TypeScript implementation providing the dashboard and coordination tools for autonomous agent peer review.

## Current Status
- **Digital Twin Documentation:** Initialized. See `AGENTS.md`, `CONCEPT.md`, `SPEC.md`, and `API_CONTRACT.md`.
- **Architecture:** Component-driven UI with centralized state management.

## Quick Start
```bash
npm install
npm run dev
```
